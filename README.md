# AI Thumbnail Generator

An AI-powered thumbnail generation platform that automatically creates high-quality thumbnails using the Gemini API. The system allows users to generate, store, and manage thumbnails efficiently with secure authentication and scalable backend architecture.

## 🚀 Features

- AI-powered thumbnail generation using Gemini API
- Structured prompt processing for consistent output
- Secure authentication with JWT
- MongoDB database for storing generated assets
- Asynchronous backend processing for concurrent requests
- Optimized middleware reducing API response time
- Persistent user sessions
- Scalable MERN stack architecture

## 🛠 Tech Stack

Frontend:
- React.js
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Authentication:
- JWT (JSON Web Token)

AI Integration:
- Google Gemini API

Deployment:
- Vercel (Frontend)
- Render (Backend)
